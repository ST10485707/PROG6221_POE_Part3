## Features Implemented

### 1. Task Assistant with MySQL Database
- Add cybersecurity tasks with descriptions and reminders
- View all tasks (saved permanently in MySQL)
- Tasks persist even after closing the application

### 2. Cybersecurity Quiz
- 11 questions covering cybersecurity topics
- Multiple choice format (1-4)
- Immediate feedback after each answer
- Final score with motivational message

### 3. Activity Log
- Tracks all user actions (task added, quiz started, etc.)
- Shows last 10 actions

### 4. NLP Simulation (Keyword Recognition)
- Recognises natural language variations:
  - Password, 2FA, authentication
  - Scam, fraud, phishing, suspicious links
  - Privacy, data breach, social engineering
  - Malware, virus, ransomware, safe browsing

### 5. GUI Design
- Dark theme with coloured text
- Chat display with auto-scroll
- Quick action buttons

## How to Run the Program

1. Clone the repository
2. Open `PROG6221_POE_Part3.slnx` in Visual Studio
3. Ensure MySQL is installed and running
4. Update password in `DatabaseHelper.cs` if needed
5. Press F5 to run
